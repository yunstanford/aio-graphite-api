aio-graphite-api
================

Highlights:

Dev mode.

async graphite-api written in Python3.5+.

Designed for and focused on fast and large volumes of queries.

No Graph Rendering.

No DB dependency.


-----
Build
-----

.. code::

	./uranium build

----
dev
----

.. code::

	./uranium dev

config
------

All configs should put into config folder.
